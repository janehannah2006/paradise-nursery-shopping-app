# Paradise Nursery Shopping Application

Paradise Nursery Shopping Application is an online plant shopping website built using React and Redux.

## Features

- Browse different categories of plants
- Add plants to shopping cart
- Increase and decrease quantities
- Remove items from cart
- Dynamic cart item counter
- Responsive shopping cart page

## Technologies Used

- React
- Redux Toolkit
- CSS

## Purpose

This project demonstrates state management and e-commerce cart functionality using React and Redux.
